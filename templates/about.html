Project Overview

{{ brand_name }} is a comprehensive machine learning platform designed to predict diabetes risk using 7 advanced data mining algorithms. Built as an educational research project, it demonstrates the power of ensemble learning in medical diagnostics.

Methodology

1
Data Collection

Upload your own medical dataset or use the default Pima Indians dataset with 768 samples.

2
Data Preprocessing

StandardScaler normalization ensures all features contribute equally regardless of scale.

3
Feature Engineering

All 8 medical parameters used as features: Pregnancies, Glucose, BloodPressure, SkinThickness, Insulin, BMI, DiabetesPedigreeFunction, Age.

4
Model Training

7 different ML algorithms trained and evaluated with 5-fold cross-validation.

5
Model Selection

Best model selected automatically based on F1-Score for optimal precision-recall balance.

6
Deployment

Interactive Flask web application with user authentication and persistent prediction history.
